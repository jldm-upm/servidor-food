# Servidor de Info de Sostenibilidad (servidor-food)

Proporciona dos APIS:
* Acceso a la información de los productos
* Manejo de cuentas de usuario

* Instalación
** Instalar las dependencias
 #+BEGIN_SRC sh

 npm install

 #+END_SRC

** Iniciar el servidor
 #+BEGIN_SRC sh

 npm start

 #+END_SRC

** Ejecutar tests
 #+BEGIN_SRC sh

 npm test

 #+END_SRC


** Contar líneas de código
 #+BEGIN_SRC sh

 npm run locs

 #+END_SRC
